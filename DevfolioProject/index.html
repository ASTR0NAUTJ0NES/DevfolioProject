Radford Senior Project - Bacterial Simulator

I worked with a team using the Agile process to coordinate, execute, and maintain a complex software project. The languages used were PHP and JavaScript via the Laravel Framework. Our team started with adopting a pre-existing project from a previous semester and learning the languages and framework used over the first couple of weeks to get up to speed on how everything worked. We then learned what all their original code was doing and how to change what we needed to.
